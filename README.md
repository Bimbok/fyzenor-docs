# Fyzenor Documentation Website

This is a modern, responsive, and interactive React + TypeScript + Vite documentation application for **Fyzenor (v4.0.0)**.

## 🚀 Quick Start

### 1. Install Dependencies
Run this command inside the project directory:
```bash
npm install
```

### 2. Run Development Server
Start the dev server locally:
```bash
npm run dev
```

### 3. Build for Production
Compile the optimized static build assets:
```bash
npm run build
```
The compiled files will be outputted to the `dist/` directory, which can be hosted directly on **GitHub Pages**, Vercel, Netlify, or any static web host.

## 🛠️ Features Included
*   **Interactive Terminal Sandbox**: Test CLI arguments in a simulated shell interface.
*   **Virtual Keyboard Shortcut Map**: Interactive keycap triggers mapping all 35+ shortcuts in real-time.
*   **TUI Theme Builder**: Real-time ncurses theme generator exporting custom `colors.fz` configurators.
*   **Multithreaded SVG diagram**: Visualizing core thread synchronization.
*   **Fully Responsive & Mobile Optimized**: Hamburger sidebar, backdrop toggles, and scaled layouts.
